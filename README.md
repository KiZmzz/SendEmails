# SendEmailsTool (批量发信工具)

一款基于 `PyQt5` 和 `QFluentWidgets` 构建的现代化企业级批量邮件发送桌面应用。它拥有极致流畅的 Fluent Design 设计，内置微软 OAuth2.0 无感授权，并提供所见即所得的 Web 级富文本排版能力。

## ✨ 核心特性

- 🎨 **现代化 UI 设计**：基于 `QFluentWidgets`，界面美观、极简，完美适配深色/浅色模式，交互平滑如丝。
- 📝 **Web 级富文本编辑器**：底层深度集成 **wangEditor 5**，支持极其丰富的图文混排、多级列表、格式刷等企业级排版体验；图片支持免服务器直转 Base64 内嵌。
- 🔐 **系统级高强度加密**：全面接入 `keyring`。您的账号密码以及微软授权 Token 都不会在本地明文留存，而是直接被**强加密保存在操作系统的原生凭据管理器中**（macOS Keychain / Windows 凭据管理器）。
- 🤖 **支持微软 Graph API**：完美支持 Outlook / Microsoft 365 邮箱发送。不仅支持 OAuth2.0 授权，还可以本地永久记忆 Token，实现**一键免重登发送**！
- 📜 **强大的模板管理系统**：支持将常用发信话术连带排版、颜色、各种富文本样式，一键保存为模板。下次发信下拉框一秒加载，所见即所得。
- 📑 **智能收件人提取**：支持从文本段落中自动使用正则提取海量邮箱，同时也支持导入 Excel 文档批量载入。
- 🛡️ **防风控与多线程**：底层具备完整的防伪造 `Message-ID` 及发信延迟机制（防阿里企业邮限流等），支持多线程异步发件，彻底告别界面卡死。

## 🚀 运行环境

- Python 3.8+ 
- 支持 macOS 和 Windows 双平台

## 📦 快速安装与启动

1. **克隆或下载本项目源码**

2. **安装核心依赖库**
   使用 `pip` 安装所需的全部依赖：
   ```bash
   pip install -r requirements.txt
   ```
   *注意：如果在国内下载缓慢，建议挂载清华源（`-i https://pypi.tuna.tsinghua.edu.cn/simple`）。*

3. **启动程序**
   ```bash
   python sendEmails.py
   ```

## 🛠️ 主要模块说明

### 1. 账号管理
- **SMTP 模式**：输入普通的邮箱账号及密码/授权码即可。支持阿里邮箱、QQ、网易、Gmail 等常见服务商。
- **Graph 模式 (Outlook)**：需要输入 Azure 平台申请的 `Client ID`，点击“微软设备码登录”授权。授权成功后建议点击**“保存当前配置”**，以后再次发送将实现全自动底层登录，彻底免去重新打开浏览器确认授权的烦恼！

### 2. 邮件内容模板
- **编写邮件**：完全遵循现代化办公软件习惯，可加粗、换色、插图。
- **存为模板**：输入名字，点击右侧保存按钮。下次哪怕重启电脑，也能在顶部的下拉框中一键唤出完整的排版模板。

### 3. 发送列表管理
可以在“收件人与发送”面板中贴入任何杂乱的内容，程序会自动过滤并揪出所有合法的邮箱地址。点击**开始发送**后，下方的进度条会实时反馈每一封的投递状态。

## ⚠️ 注意事项

- **关于图片插入**：项目底层做过深层优化，单张 `< 10MB` 的图片插入时，会直接强制转化为 `Base64` 编码内嵌于邮件中，不会导致对方收不到图片，也免去了您自建图床服务器的烦恼。
- 如果初次运行提示 `PyQtWebEngine` 环境冲突或报错，请仔细检查当前 Python 环境内是否串用了 `PyQt6` 相关的旧文件。
